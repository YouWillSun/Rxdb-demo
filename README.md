# Rxdb-Demo

**template：angular-electron**

说明：此 demo 包含对 `Rxdb` 数据库的增删改查操作，以及数据的导出与导出。

 - 依赖下载： `npm install`

 - 项目运行： `npm start`

 - 项目构建： `npm run electorn:build`

分支说明：`parse_excel`

 - 此分支主要为解析`Excel`模板，并把解析出来的数据导入数据库。
